# Marchand Lab Systems Serology Excel Data parser

Data are supplied in a wide-format, Excel file with several
worksheets.  This R-script,
`parse-Marchand-Sharma_Systems-Serology.R`, parses the Excel file into
a single long-format CSV file that uses the controlled vocabulary
developed for this INCENTIVE project. See the [Controlled-Vocabulary
repo](https://github.com/INCENTIVE-Grant/Controlled-Vocabulary) in
this same INCENTIVE-Grant project.

The data files, both Excel and CSV, associated with this parser
are located on Zenodo:  https://doi.org/10.5281/zenodo.19229040 .
